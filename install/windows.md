# Instalação no Windows

## Modo fácil

Mesmo que o tko funcione no windows, o sistema operacional não é o mais recomendado para programação. Se você está começando, considere instalar o [WSL](https://docs.microsoft.com/pt-br/windows/wsl/install) e usar o ubuntu ou outra distribuição linux.

```bash
wsl --install
```

Depois instale o tko e os compiladores como se estivesse no linux [LINK](linux.md).

## Instalando o Python sem WSL

Primeiro baixe e instale o python: [https://www.python.org/downloads/](https://www.python.org/downloads/).

## Instalando o TKO

Depois abra o power shell **COMO ADMINISTRADOR** e execute o comando:

```bash
pip install tko
```
